Les balises

Voici quelques-unes des balises HTML les plus utilisées:

  • <h1> - pour les titres principaux
  • <p> - pour les paragraphes
  • <a> - pour les liens
  • <img> - pour les images
  • <u> et <li> - pour les listes non ordonées
